# Real-Time Market Alert Bot 📈

A high-performance, reactive Java backend that monitors live market data (Stocks/Crypto) and triggers alerts based on configurable rules.

## 🛠️ Tech Stack
*   **Language:** Java 21
*   **Framework:** Spring Boot 3 (WebFlux / Reactive)
*   **Real-time:** WebSockets (Client & Server)
*   **State Management:** Redis (for sliding windows and deduplication)
*   **Build:** Gradle

## 🚀 Features (Planned)
1.  **Ingest:** Connect to live WebSocket feeds (e.g., Binance, Finnhub, Alpaca).
2.  **Analyze:** Sliding window analysis (e.g., "Price dropped 5% in 5 minutes").
3.  **Alert:** Send notifications via Telegram/Discord.
